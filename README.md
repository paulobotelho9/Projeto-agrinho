# Agro sustetável

📖 Sobre o projeto

O Agro Sustentável é uma plataforma interativa desenvolvida para conscientizar sobre a importância do agronegócio aliado à preservação do meio ambiente.

O site apresenta uma narrativa dinâmica onde o usuário toma decisões críticas sobre o uso de recursos naturais. Cada escolha gera consequências imediatas, ilustrando o impacto de práticas sustentáveis versus métodos inadequados de produção.

🎯 Objetivo
Educação Ambiental: Mostrar a importância do agronegócio para a sociedade de forma equilibrada.
Conscientização: Demonstrar as consequências do uso inadequado dos recursos naturais.
Engajamento: Incentivar práticas sustentáveis por meio de uma interface dinâmica e educativa.

🛠️ Tecnologias utilizadas:
O projeto foi construído utilizando tecnologias web fundamentais:

HTML5: Estruturação do conteúdo e das etapas da história.

CSS3: Estilização visual e responsividade.

JavaScript: Lógica de alternância entre etapas e gerenciamento das ramificações da história.

⚙️ Funcionamento
O sistema opera como uma máquina de estados simples:

O usuário inicia a jornada na tela de introdução.
Em cada etapa, são apresentadas opções de ação.
O JavaScript manipula o DOM para exibir o próximo cenário com base na escolha feita.
Cada decisão influencia o desfecho, levando a múltiplos finais possíveis.
🌿 Temas abordados
Sustentabilidade e preservação ambiental.
Uso consciente de recursos hídricos e do solo.
Equilíbrio entre desenvolvimento econômico e ecossistema.
Produção agrícola moderna e responsável.

📚 Projeto educacional
Este projeto foi desenvolvido com foco em atividades pedagógicas, utilizando o tema Agrinho – Agro Sustentável para fomentar a reflexão sobre o futuro da produção de alimentos e a segurança ambiental.
